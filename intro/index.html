Data Acquisition

The used data of this project has its origin in three different sources. We used two APIs and received some data with web crawling as well.

Spotify API

We used the Spotify API to collect the song Features of the different Tracks. To read more about the spotify API you can follow the link below.

Click here for more Information about the Spotify API!

Media Traffic - Web Crawling

We used mediatraffic to get information about chart placement. In doing so, we chose to use the year-end charts. About the song information, we fetched the related song features from the Spotify API.

Click here for more Information about the Mediatraffic Website!

Genius API

We used the Genius API to get the lyrics.

Click here for more Information about the Genius API!

Data Storage

All data was stored in .json files. We used different folders to sort the song rankings by their source (Spotify Top Hits or Media Traffic). The lyrics for these songs ended up in a separate folder.
